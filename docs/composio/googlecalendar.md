# Google Calendar — Composio Actions (Official)

Source: https://docs.composio.dev/toolkits/googlecalendar

## Actions

| Action | Description |
|--------|-------------|
| `GOOGLECALENDAR_FIND_EVENT` | Find/search events |
| `GOOGLECALENDAR_CREATE_EVENT` | Create a new event |
| `GOOGLECALENDAR_UPDATE_EVENT` | Update an existing event |
| `GOOGLECALENDAR_PATCH_EVENT` | Partially update an event |
| `GOOGLECALENDAR_DELETE_EVENT` | Delete an event |
| `GOOGLECALENDAR_QUICK_ADD` | Create event from natural language text |
| `GOOGLECALENDAR_EVENTS_GET` | Get a specific event by ID |
| `GOOGLECALENDAR_EVENTS_LIST` | List events (alternative to FIND_EVENT) |
| `GOOGLECALENDAR_EVENTS_MOVE` | Move event between calendars |
| `GOOGLECALENDAR_LIST_CALENDARS` | List available calendars |
| `GOOGLECALENDAR_GET_CALENDAR` | Get calendar metadata |
| `GOOGLECALENDAR_FIND_FREE_SLOTS` | Find free/busy time slots |
| `GOOGLECALENDAR_REMOVE_ATTENDEE` | Remove an attendee from event |
