# Todoist — Composio Actions Reference

## Toolkit: `todoist` (managed app ✓)

### Actions mapeadas

| Route | Action | Descrição |
|-------|--------|-----------|
| `GET /projects` | `TODOIST_GET_ALL_PROJECTS` | Listar projetos |
| `GET /tasks` | `TODOIST_GET_ALL_TASKS` | Listar tarefas |
| `POST /tasks` | `TODOIST_CREATE_TASK` | Criar tarefa |
| `POST /tasks/{id}/close` | `TODOIST_CLOSE_TASK` | Completar tarefa |
