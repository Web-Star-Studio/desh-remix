# Google Sheets — Composio Actions (Official)

Source: https://docs.composio.dev/toolkits/googlesheets

## Actions

| Action | Description |
|--------|-------------|
| `GOOGLESHEETS_CREATE_GOOGLE_SHEET1` | Create a new spreadsheet |
| `GOOGLESHEETS_GET_SPREADSHEET_INFO` | Get spreadsheet metadata |
| `GOOGLESHEETS_VALUES_GET` | Read values from a range |
| `GOOGLESHEETS_VALUES_UPDATE` | Update values in a range |
| `GOOGLESHEETS_SPREADSHEETS_VALUES_APPEND` | Append values to a range |
| `GOOGLESHEETS_BATCH_GET` | Batch read multiple ranges |
| `GOOGLESHEETS_BATCH_UPDATE` | Batch update multiple ranges |
| `GOOGLESHEETS_CLEAR_VALUES` | Clear values from a range |
