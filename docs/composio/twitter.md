# Twitter/X — Composio Actions Reference

## Toolkit: `twitter` (managed app ✓)

### Actions mapeadas

| Route | Action | Descrição |
|-------|--------|-----------|
| `POST /tweets` | `TWITTER_CREATION_OF_A_POST` | Criar tweet |
| `GET /me` | `TWITTER_USER_LOOKUP_ME` | Perfil do usuário |
| `GET /bookmarks` | `TWITTER_BOOKMARKS_BY_USER` | Bookmarks |
| `GET /timeline` | `TWITTER_USER_TIMELINE` | Timeline |
