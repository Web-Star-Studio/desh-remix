# Google Meet — Composio Actions (Official)

Source: [https://docs.composio.dev/toolkits/googlemeet](https://docs.composio.dev/toolkits/googlemeet)

## Actions


| Action                               | Description                |
| ------------------------------------ | -------------------------- |
| `GOOGLEMEET_CREATE_SPACE`            | Create a new meeting space |
| `GOOGLEMEET_GET_SPACE`               | Get meeting space details  |
| `GOOGLEMEET_LIST_CONFERENCE_RECORDS` | List conference records    |
