# Gmail — Composio Actions (Official)

Source: https://docs.composio.dev/toolkits/gmail

## Actions

| Action | Description |
|--------|-------------|
| `GMAIL_SEND_EMAIL` | Send an email |
| `GMAIL_CREATE_EMAIL_DRAFT` | Create an email draft |
| `GMAIL_REPLY_TO_THREAD` | Reply to a thread |
| `GMAIL_FORWARD_MESSAGE` | Forward an email message |
| `GMAIL_FETCH_EMAILS` | Fetch emails from inbox |
| `GMAIL_FETCH_MESSAGE_BY_MESSAGE_ID` | Fetch a single message by ID |
| `GMAIL_FETCH_MESSAGE_BY_THREAD_ID` | Fetch all messages in a thread |
| `GMAIL_LIST_THREADS` | List email threads |
| `GMAIL_GET_ATTACHMENT` | Get email attachment data |
| `GMAIL_ADD_LABEL_TO_EMAIL` | Add/remove labels on a message |
| `GMAIL_MODIFY_THREAD_LABELS` | Batch modify labels on messages |
| `GMAIL_BATCH_MODIFY_MESSAGES` | Batch modify messages |
| `GMAIL_BATCH_DELETE_MESSAGES` | Batch delete messages permanently |
| `GMAIL_MOVE_TO_TRASH` | Move message to trash |
| `GMAIL_DELETE_MESSAGE` | Permanently delete a message |
| `GMAIL_UNTRASH_MESSAGE` | Remove message from trash |
| `GMAIL_LIST_LABELS` | List all labels |
| `GMAIL_CREATE_LABEL` | Create a new label |
| `GMAIL_DELETE_LABEL` | Delete a label |
| `GMAIL_LIST_DRAFTS` | List all drafts |
| `GMAIL_GET_DRAFT` | Get a specific draft |
| `GMAIL_SEND_DRAFT` | Send an existing draft |
| `GMAIL_GET_PROFILE` | Get user's Gmail profile |
| `GMAIL_LIST_HISTORY` | List history of changes |
| `GMAIL_GET_CONTACTS` | Get user's Google contacts |
| `GMAIL_GET_PEOPLE` | Get people by resource name |
| `GMAIL_SEARCH_PEOPLE` | Search contacts/people |
