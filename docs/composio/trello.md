# Trello — Composio Actions Reference

## Toolkit: `trello` (managed app ✓)

### Actions mapeadas

| Route | Action | Descrição |
|-------|--------|-----------|
| `GET /boards` | `TRELLO_GET_MEMBERS_BOARDS_BY_ID_MEMBER` | Listar boards do usuário |
| `GET /boards/{id}/lists` | `TRELLO_GET_BOARDS_LISTS_BY_ID_BOARD` | Listas de um board |
| `GET /lists/{id}/cards` | `TRELLO_GET_LISTS_CARDS_BY_ID_LIST` | Cards de uma lista |
| `POST /cards` | `TRELLO_POST_CARDS` | Criar card |
