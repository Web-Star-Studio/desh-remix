# Dropbox — Composio Actions Reference

## Toolkit: `dropbox` (managed app ✓)

### Actions mapeadas

| Route | Action | Descrição |
|-------|--------|-----------|
| `GET /files/list` | `DROPBOX_LIST_FOLDER` | Listar arquivos |
| `GET /files/search` | `DROPBOX_SEARCH_FILES_V2` | Buscar arquivos |
| `POST /files/upload` | `DROPBOX_UPLOAD` | Upload de arquivo |
| `GET /files/download` | `DROPBOX_DOWNLOAD` | Download de arquivo |
