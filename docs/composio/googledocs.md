# Google Docs — Composio Actions (Official)

Source: https://docs.composio.dev/toolkits/googledocs

## Actions

| Action | Description |
|--------|-------------|
| `GOOGLEDOCS_CREATE_DOCUMENT` | Create a new document |
| `GOOGLEDOCS_GET_DOCUMENT_BY_ID` | Get document by ID |
| `GOOGLEDOCS_GET_DOCUMENT_PLAINTEXT` | Get document as plain text |
| `GOOGLEDOCS_UPDATE_EXISTING_DOCUMENT` | Update document content |
| `GOOGLEDOCS_SEARCH_DOCUMENTS` | Search for documents |
