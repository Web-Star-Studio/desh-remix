# LinkedIn — Composio Actions (Official)

Source: https://docs.composio.dev/toolkits/linkedin

## Actions

| Action | Description |
|--------|-------------|
| `LINKEDIN_CREATE_LINKED_IN_POST` | Create a LinkedIn post |
| `LINKEDIN_CREATE_POST_LINKEDIN` | Create post (alternate) |
| `LINKEDIN_GET_USER_PROFILE` | Get authenticated user profile |
