# Microsoft Teams — Composio Actions Reference

## Toolkit: `teams` → `microsoftteams` (managed app)

### Actions mapeadas

| Route | Action | Descrição |
|-------|--------|-----------|
| `GET /channels` | `MICROSOFTTEAMS_LIST_CHANNELS` | Listar canais |
| `POST /messages/send` | `MICROSOFTTEAMS_SEND_CHANNEL_MESSAGE` | Enviar mensagem |
| `GET /teams` | `MICROSOFTTEAMS_LIST_JOINED_TEAMS` | Listar times |
