# Outlook — Composio Actions Reference

## Toolkit: `outlook` (Microsoft, managed app)

### Actions mapeadas

| Route | Action | Descrição |
|-------|--------|-----------|
| `GET /messages` | `OUTLOOK_LIST_MESSAGES` | Listar emails |
| `POST /messages/send` | `OUTLOOK_SEND_EMAIL` | Enviar email |
| `GET /folders` | `OUTLOOK_LIST_MAIL_FOLDERS` | Listar pastas |
