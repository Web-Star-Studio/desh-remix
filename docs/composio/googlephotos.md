# Google Photos — Composio Actions (Official)

Source: https://docs.composio.dev/toolkits/googlephotos

## Actions

| Action | Description |
|--------|-------------|
| `GOOGLEPHOTOS_LIST_ALBUMS` | List photo albums |
| `GOOGLEPHOTOS_GET_ALBUM` | Get album details |
| `GOOGLEPHOTOS_CREATE_ALBUM` | Create a new album |
| `GOOGLEPHOTOS_SEARCH_MEDIA_ITEMS` | Search for media items |
| `GOOGLEPHOTOS_LIST_MEDIA_ITEMS` | List media items |
| `GOOGLEPHOTOS_GET_MEDIA_ITEM` | Get a specific media item |
